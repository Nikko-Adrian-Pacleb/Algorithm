About Sort Algorithm

Sort Algorithm is a way to reorganize a set of data in a structure that they are in order like alphabetical, greater values, etc.

Insertion Sort

Insertion sort works by going through the list from the first to last. In each index we will move the object back until the value before it is lower(or higher depending on sort order) than the index's value. The most common example of Insertion sort is sorting a deck of cards.

Selection Sort

Selection Sort works by going through the list and look for the smallest(or biggest depending on the sort order) and place them on the start of the list. The algorithm then increase the index of one then start looking and placing the smallest on that index.

Bubble Sort

Bubble Sort starts by having two index and compare the value of those two and swap them depending on the sort order. Then after the smalles(or biggest depending on sort order) is on its place, the index then change a space away from that index.
